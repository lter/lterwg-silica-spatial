# QAQC Tools

QAQC scripts are grouped by what they check.

- `final_checks/`: final full-dataset, ESOM, LULC, FNConc, snow/permafrost, and site time-series diagnostics.
- `readiness/`: final-run gap, coverage, ESOM dynamic, and missing-polygon readiness checks.
- `extraction_checks/`: raw dynamic coverage, HydroSHEDS assignment, and region-specific extraction checks.
- `comparisons/`: old-vs-new dataset comparison scripts.

Write new QAQC scripts to the narrowest matching subfolder.
